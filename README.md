# 📄 Resume Templates for Freshers (India)

**Free, ATS-friendly resume templates** designed for Indian freshers — B.Com, MBA, CA, Engineering graduates. Get shortlisted, not filtered out.

---

## 🚫 Why Most Fresher Resumes Get Rejected

```
❌ Fancy designs that ATS can't read
❌ "Objective: To work in a reputed organization..." (generic)
❌ 3-page resume for 0 years experience
❌ No keywords matching the job description
❌ Skills section with "Hard working, Team player" (means nothing)
```

---

## ✅ What Makes a Great Fresher Resume

```
✅ 1 page only (strict rule for freshers)
✅ ATS-compatible format (simple, clean, parseable)
✅ Strong action verbs (Built, Analyzed, Created, Led)
✅ Quantified achievements (Increased X by 20%)
✅ Keywords from the job description
✅ Projects > Hobbies (show what you've DONE)
```

---

## 📋 Resume Structure (Recommended Order)

```
┌────────────────────────────────────────┐
│  NAME & CONTACT INFO                   │
│  Phone | Email | LinkedIn | City       │
├────────────────────────────────────────┤
│  PROFESSIONAL SUMMARY (2-3 lines)      │
│  Who you are + what you bring          │
├────────────────────────────────────────┤
│  EDUCATION                             │
│  Degree, University, Year, CGPA/%      │
├────────────────────────────────────────┤
│  PROJECTS (Most Important Section!)    │
│  What you built + tech used + impact   │
├────────────────────────────────────────┤
│  SKILLS                                │
│  Technical + Domain + Tools            │
├────────────────────────────────────────┤
│  CERTIFICATIONS (if any)               │
├────────────────────────────────────────┤
│  EXTRACURRICULARS (keep brief)         │
└────────────────────────────────────────┘
```

## 🎯 Power Words for Resumes

### Action Verbs (Use These!)
```
Analyzed | Built | Created | Developed | Designed
Implemented | Improved | Increased | Led | Managed
Optimized | Organized | Prepared | Reduced | Streamlined
```

### Words to AVOID
```
❌ Responsible for... (passive)
❌ Helped with... (vague)  
❌ Hard working (everyone says this)
❌ Team player (prove it, don't say it)
❌ Self-motivated (show through projects)
```

## 📊 ATS Tips

| Do | Don't |
|-----|--------|
| Use simple formatting | Use tables, graphics, columns |
| Standard section headings | Creative section names |
| .docx or .pdf format | .jpg or designed PDFs |
| Match keywords from JD | Use different terminology |
| Spell out acronyms once | Assume reader knows abbreviations |
